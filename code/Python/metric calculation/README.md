## About The Project

Metrics for quality evaluation of image fusion tasks are collected.

<p align="right">(<a href="#readme-top">back to top</a>)</p>

<!-- GETTING STARTED -->
## Getting Started

### Prerequisites
- opencv-python
- pytorch
- scikit-image
- tqdm

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/HangYou-Nick/FusionEval.git
   ```

<p align="right">(<a href="#readme-top">back to top</a>)</p>



<!-- USAGE EXAMPLES -->
## Usage
Change root paths of datasets in `main.py` and metrics, and execute the following code
```sh
python main.py
```
